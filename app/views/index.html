Share information for your Blue Badge application v0.1

{{releaseVersion | log }}

Created: 16 April 2019

Changes:

  • prototype created

Research showed:

  • some users incorrectly thought the service was to apply for a Blue Badge
  • some users said they would drop out if they had to create an ID account
  • users are able to select their council from 4 radio buttons
  • after entering their postcode, some users expected to see a list of addresses to select – but address isn't needed
  • users understand what will happen to their PIP or DLA when they click the 'Send to your council' button
  • on the completed page all users were confused about what the reference code was for and what they had to do next

Caveats:

  • we don't know yet what the best way for users to prove who are they are is (Government Gateway is just one example)
  • private beta will cover information sharing for a small number of councils
  • this version is for a stand-alone service for citizens who apply for a Blue Badge using their local council's online service or paper form
  • we need to test a version that is part of a seamless Apply for a Blue Badge service
  • this version is the happy path user journey – we also need to test an unhappy path journey
