v1 {{releaseVersion | log }}

Created: 16 April 2019

Changes:

  • prototype created

Research showed:

  • some users wrongly thought the service is to apply for a Blue Badge
  • some users would drop out if they had to create an ID account
  • all users were able to select their council from 4 radio buttons
  • some users expected to see a list of addresses to select after entering a postcode– but address isn't needed
  • all users understand what will happen to their PIP or DLA info when they click 'Send to your council'
  • all users were confused about what the reference code is for and what to do next on the last page

Caveats:

  • we don't know yet what the best way for users to prove who they are is (Government Gateway is just one example)
  • private beta will cover information sharing for a small number of councils
  • this version is for a stand-alone service for citizens who apply for a Blue Badge using their local council's online service or paper form
  • we need to test a version that is part of a seamless Apply for a Blue Badge service
  • this version is the happy path user journey – we also need to test an unhappy path journey
